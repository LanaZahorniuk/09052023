

#!/bin/bash
 
touch /root/lana/output.txt
 
df -h | awk '{print$4}' | tail -2 | sed -n '2p' | sed 's/М/kb/g' >> /root/lana/>



создает пустой файл с именем output.txt в директории /root/lana/.
Команда df используется для вывода информации о доступных дисковых пространствах. Параметр -h указывает, чтобы выводить информацию в удобочитаемом формате с единицами измерения (KB, MB, GB и т. д.).
awk '{print$4}': Команда awk используется для обработки текстовых данных в столбцах (полях). В данном случае, она выводит только четвертое поле (столбец) из вывода команды df -h. Это поле представляет доступное дисковое пространство в удобочитаемом формате.
мы хотим получить последние две строки вывода, которые представляют доступное дисковое пространство для двух различных разделов или файловых систем.
вывести только вторую строку из вывода tail -2. Таким образом, получаем вторую строку, которая представляет доступное дисковое пространство для второго раздела или файловой системы.
's/М/kb/g' указывает заменить все вхождения символа "М" на "kb" во второй строке (которая представляет доступное дисковое пространство).
и дозапись в /root/lana
